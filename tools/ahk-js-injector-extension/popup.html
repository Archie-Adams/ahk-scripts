This extension injects the contents of the clipboard as JavaScript into the active tab when the keyboard shortcut Ctrl+Shift+ Insert is pressed.

This is designed to be used by AHK scripts to interact with webpages by replacing the contents of the clipboard with JavaScript, sending the complex shortcut, then replacing the contents of the clipboard.

An example implementation in AHK (Clicking the Subscriptions button on youtube.com when Ctrl+Shift+ S is pressed): 

    
      ^+s::
      temp := clipboardall
      clipboard := "document.querySelectorAll('[title=""Subscriptions""]')[0].click();"
      Send, ^+{Insert}
      clipboard := temp
      return

When using many hotkeys the following function may be preferable to use.

    
      sendJS(url, js)
      {
        temp := clipboardall
        clipboard := %url% %js%
        Send, ^+{Insert}
        clipboard := temp
      }

      s::
      sendJS("", "document.querySelectorAll('[title=""Subscriptions""]')[0].click();")
      return
