# german-umlauts.ahk

Allows for easier typing of german umlauts via hotstrings.  

---

## Features

### Expansions

| Hotstring | Expands to |
| --------- | ---------- |
| ``a       | ä          |
| ~~a       | Ä          |
| ``u       | ü          |
| ~~u       | Ü          |
| ``o       | ö          |
| ~~o       | Ö          |
| ``s       | ß          |
