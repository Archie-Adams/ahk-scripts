# german-umlauts.ahk

Allows for easier typing of german umlauts via hotstrings.  

---

## Features

| Command                    | Description |
| -------------------------- | ----------- |
| R-ALT + R-CTRL + a         | ä           |
| R-ALT + R-CTRL + SHIFT + a | Ä           |
| R-ALT + R-CTRL + u         | ü           |
| R-ALT + R-CTRL + SHIFT + u | Ü           |
| R-ALT + R-CTRL + o         | ö           |
| R-ALT + R-CTRL + SHIFT + o | Ö           |
| R-ALT + R-CTRL + s         | ß           |
